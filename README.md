# OneXFly Performance Plugin for Decky Loader

Плагин для включения/отключения performance режима на OneXFly F1 Pro.

## Функции
- Простая кнопка включения/отключения performance режима
- Автоматическое применение настроек
- Мониторинг статуса

## Установка
1. Скопируйте папку `onexfly-performance` в `/home/deck/homebrew/plugins/`
2. Перезагрузите Decky Loader или SteamOS

## Использование
Нажмите кнопку в плагине для включения/отключения performance режима.

## Требования
- SteamOS
- Decky Loader
- OneXFly F1 Pro
